## 🚀 About This Project

**PrimeSA_Bot by Professor Sahil** is a WhatsApp Multi-Device bot built using the **Baileys library**.

It is designed for:
- ⚡ Fast performance  
- 🧠 AI chatbot features  
- 🔧 Easy customization  
- 🛠️ Beginner friendly setup  
- 🔒 Stable deployment  

---

## ⚙️ Important Configuration (VERY IMPORTANT)

### 📌 Change Bot Name & Owner

If you want to change:
- Bot name  
- Owner name  
- Branding  

👉 Go to:


config.js


And edit:

```js
module.exports = {
    BOT_NAME: "PrimeSA_Bot",
    OWNER_NAME: "Professor Sahil"
}
🚀 Installation Guide (STEP BY STEP)
1️⃣ Clone Repository
git clone https://github.com/sahillume/PrimeSA_Bot.git
cd PrimeSA_Bot
2️⃣ Install Dependencies
npm install
3️⃣ Setup Configuration

Open:

config.js

Set:

BOT_NAME
OWNER_NAME
sessionID (if required)
4️⃣ Start Bot
node index.js
